## Test locally
1. Pull the repository to your local machine.

2. Create `.env` in `backend/` if not already present:
```
# Required
MONGO_URI=********

# Optional
PORT=********
JWT_SECRET=********
```
3. Run the following command:
```bash
npm run dev
```
4. Go to your browser and connect to:
```
localhost:3001
```
